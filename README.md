# Assignment Tracking & Notification System

This is a MERN Stack project designed to help students avoid missing assignment submissions through a centralized tracking and notification system.

## Features
- Staff can upload assignments
- Students can view and track assignments
- Submission status tracking (Pending / Submitted / Overdue)
- In-app notification system 
- Sound alert for important updates 
- Multi-staff assignment management
- Due date tracking with overdue highlighting 
- Filter assignments by Year and Department
- Staff submission overview dashboard

## Tech Stack
- MongoDB
- Express.js
- React.js
- Node.js

## Modules

### Staff Module
- Upload assignments
- View submission overview
- Filter by year and department

### Student Module
- View assignments
- Mark submission status
- Receive notifications

### Notification Module
- In-app alerts
- Sound alerts
- Due date reminders

## Problem Statement
Students often miss assignment deadlines due to lack of proper tracking and scattered communication.

## Solution
This system provides a centralized dashboard with notifications, tracking, and alerts to ensure timely submission of assignments.

## Unique Features
- Sound alert system 
- Overdue visual indicator 
- Centralized multi-staff dashboard
- Clean and modern UI
